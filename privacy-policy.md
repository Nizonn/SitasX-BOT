# SitasX Discord Bot Privacy Policy

Last Updated: 2026

This Privacy Policy explains how the SitasX Discord Bot handles information.

## Information Collected

The bot may collect limited information required to operate:

- Discord User IDs
- Discord Server IDs
- Channel IDs
- Music queue information
- Command usage data

## Purpose

Data is used only to:

- process music commands
- manage playback queues
- maintain bot functionality
- debug system issues

## Storage

Some operational data may be temporarily stored using systems such as:

- MySQL
- Redis

These systems help maintain queue and playback state.

## Data Sharing

SitasX does not sell or share user data with third parties.

## Third Party Services

The bot relies on services such as:

- Discord
- Lavalink
- music streaming providers

These services operate under their own privacy policies.

## Data Retention

Data is stored only as long as necessary for bot operation.

## Security

Reasonable measures are taken to protect stored data.

## Updates

This policy may be updated at any time.

## Contact

For privacy concerns please contact the developer via the support server or GitHub.
